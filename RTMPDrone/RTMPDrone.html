กรอกที่อยู่ RTMP สตรีมของโดรน เช่น rtmp://192.168.1.50/live/drone
ใช้ localhost
ชื่อ Source ในแถบ Sources ของ OBS ที่จะรับภาพนี้

ขั้นตอนการตั้งค่าใน OBS Studio:

  1. เปิดโปรแกรม OBS Studio ขึ้นมา
  2. ไปที่เมนูด้านบน Tools (เครื่องมือ) > OBS WebSocket Settings
  3. ติ๊กถูกที่ช่อง Enable WebSocket server (เปิดใช้งานเซิร์ฟเวอร์ WebSocket)
  4. จดพอร์ต (ปกติคือ 4455) และกดปุ่ม Show Connect Info เพื่อดูรหัสผ่าน
  5. นำข้อมูลมากรอกในแท็บ "ตั้งค่าการเชื่อมต่อ OBS WebSocket" ด้านบน

การทำเป็นเมนู Dock ในโปรแกรม OBS (แนะนำ!):

  1. หลังจากกรอกการตั้งค่าเสร็จแล้ว ให้กดปุ่ม สร้างลิงก์ด่วน ลิงก์เชื่อมต่อที่มีรหัสผ่านจะถูกก๊อปปี้
  2. ในโปรแกรม OBS ไปที่เมนูด้านบน Docks (แถบเครื่องมือ) > Custom Browser Docks... (ด็อคบราวเซอร์แบบกำหนดเอง...)
  3. ในช่อง Dock Name ตั้งชื่อเช่น Drone Remote
  4. ในช่อง URL ให้วางลิงก์ที่ก๊อปปี้มาลงไป แล้วกดปุ่ม Apply (นำไปใช้)
  5. โปรแกรม OBS จะเปิดหน้าต่างรีโมทนี้ขึ้นมาเชื่อมต่อกับระบบควบคุมของท่านทันทีโดยไม่ต้องเปิดเบราว์เซอร์แยก!
