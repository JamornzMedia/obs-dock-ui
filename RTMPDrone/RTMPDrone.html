เครือข่าย 1. เชื่อมต่อ Wi-Fi วงเดียวกัน

เพื่อให้อุปกรณ์ควบคุม (เช่น มือถือ/แท็บเล็ต) สามารถสื่อสารกับโปรแกรม OBS บนคอมพิวเตอร์ได้ ทั้งคอมพิวเตอร์และมือถือต้องเชื่อมต่อ Wi-Fi ตัวเดียวกัน (หรือคอมพิวเตอร์เชื่อมต่อ Wi-Fi Hotspot ที่เปิดจากมือถือเครื่องที่ใช้ควบคุม)

ฝั่งคอมพิวเตอร์ (PC) 2. วิธีหา IP ของเครื่องคอมพิวเตอร์

  1. บนระบบ Windows: กดปุ่ม Windows + R บนคีย์บอร์ด พิมพ์ cmd แล้วกด Enter เพื่อเปิด Command Prompt
  2. พิมพ์คำสั่ง ipconfig แล้วกด Enter
  3. มองหาหัวข้อ Wireless LAN adapter Wi-Fi (หากต่อ Wi-Fi) หรือ Ethernet adapter (หากต่อสายแลน)
  4. มองหาบรรทัด IPv4 Address จะเห็นตัวเลข IP เช่น 192.168.1.50 (จดหมายเลข IP นี้ไว้เพื่อนำไปกรอกบนมือถือ)
  5. บนระบบ macOS: ไปที่ การตั้งค่าระบบ (System Settings) > Network (เครือข่าย) แล้วเลือกการเชื่อมต่อที่กำลังใช้งานอยู่ จะมีหมายเลข IP แสดงชัดเจน

ฝั่งคอมพิวเตอร์ (PC) 3. ตั้งค่า OBS WebSocket Settings

  1. เปิดโปรแกรม OBS Studio
  2. ไปที่เมนูด้านบน Tools (เครื่องมือ) > OBS WebSocket Settings
  3. ติ๊กถูกที่ช่อง Enable WebSocket server (เปิดใช้งานเซิร์ฟเวอร์ WebSocket)
  4. จดจำหมายเลขพอร์ต (ปกติคือ 4455) และกดปุ่ม Show Connect Info เพื่อดูรหัสผ่าน (Server Password)
  5. ในช่อง Sources ของ OBS จะต้องมี Media Source ชื่อตรงกับที่ระบุ (เช่น DroneSource) โดยคุณสามารถกดปุ่ม "➕ สร้าง Source" ในแผงตั้งค่าด้านบนเพื่อให้ระบบสร้างใน Scene ปัจจุบันอัตโนมัติ หรือจะแอดมือเองก็ได้ (ห้ามเลือก Local File)

ฝั่งมือถือ (Mobile) 4. วิธีเปิดใช้งานและเชื่อมต่อจากโทรศัพท์

  1. ทางเลือกเปิดใช้งานหน้าเว็บ:
    • ส่งไฟล์ RTMPDrone.html ไปเปิดในบราวเซอร์มือถือตรงๆ
    • หรือโฮสต์ขึ้นเว็บเซิร์ฟเวอร์ฟรี (เช่น GitHub Pages) เพื่อความสะดวกในการเปิดและใช้ฟีเจอร์กล้องสแกน QR
  2. กดเมนู "ตั้งค่าการเชื่อมต่อ OBS WebSocket" ด้านบน
  3. กรอกข้อมูลที่ได้จากคอมพิวเตอร์:
    • OBS IP/Host: ใส่ IP คอมพิวเตอร์จากข้อ 2 (เช่น 192.168.1.50) (ห้ามใช้ 127.0.0.1 บนมือถือ เพราะจะอ้างอิงตัวมือถือเอง)
    • Port: 4455 (ตามที่ตั้งใน OBS)
    • รหัสผ่าน OBS WebSocket: ใส่รหัสผ่านที่คัดลอกมาจาก OBS
  4. กดปุ่ม "เชื่อมต่อ & บันทึก" เพื่อจำการตั้งค่า (แถบสถานะด้านบนจะเปลี่ยนเป็นสีเขียว "เชื่อมต่อ OBS สำเร็จ")

💡 เคล็ดลับทางลัด (สร้างลิงก์ด่วน): บน PC คุณสามารถกรอก IP เครื่อง PC, พอร์ต, รหัสผ่าน ไว้ล่วงหน้า แล้วกดปุ่ม "สร้างลิงก์ด่วน" จากนั้นส่งลิงก์ยาวๆ ที่ถูกคัดลอกนั้นไปเปิดในมือถือ หน้าเว็บมือถือจะตั้งค่าและเชื่อมต่อให้ทันที!

⚠️ หมายเหตุเรื่องกล้อง: บราวเซอร์บนมือถือจะอนุญาตให้ใช้กล้องสแกน QR Code เฉพาะหน้าเว็บที่ทำงานผ่านโปรโตคอล HTTPS:// (เช่น โฮสต์บนอินเทอร์เน็ต) หรือ localhost เท่านั้น หากเปิดไฟล์ธรรมดา (file://) หรือ HTTP กล้องอาจจะไม่ทำงาน และต้องใช้วิธีก๊อปปี้ลิงก์ RTMP มาวางเองครับ

ฝั่งมือถือ (Mobile) 5. ส่งสตรีมโดรนเข้า OBS

  1. นำลิงก์ RTMP สตรีมของโดรนที่ได้จากแอปสตรีมหรือรีโมทโดรน มาวางในช่อง ลิงก์สตรีมวิดีโอ (เช่น rtmp://192.168.1.100/live/drone)
  2. หรือกดปุ่ม 📷 สแกน QR Code บนหน้าควบคุมนี้ แล้วใช้กล้องสแกนคิวอาร์โค้ดของลิงก์สตรีม
  3. กดปุ่ม "ส่งภาพฉายใน OBS" ปุ่มสีน้ำเงินด้านบน ลิงก์จะถูกส่งไปฉายบน OBS ทันที!
