📝 วิธีเปิดใช้งาน WebSocket ใน OBS (ทำเฉพาะครั้งแรก):
  1. ในโปรแกรม OBS Studio ไปที่เมนูด้านบน Tools (เครื่องมือ) > OBS WebSocket Settings
  2. ติ๊กถูกที่ช่อง Enable WebSocket server (เปิดใช้งานเซิร์ฟเวอร์ WebSocket)
  3. จดพอร์ต (ปกติคือ 4455) และกดปุ่ม Show Connect Info เพื่อดูรหัสผ่าน (Server Password) แล้วนำมากรอกด้านบน

เครือข่าย 1. เชื่อมต่อ Wi-Fi เดียวกัน

คอมพิวเตอร์ที่เปิด OBS และรีโมทโดรน (หรือโทรศัพท์มือถือที่คุมโดรน) ต้องเชื่อมต่อ Wi-Fi ตัวเดียวกัน (หรือเปิด Hotspot จากมือถือแล้วเอาคอมพิวเตอร์มาเชื่อมต่อร่วมกันก็ได้)

ฝั่งคอมพิวเตอร์ (PC) 2. วิธีค้นหา IP ของคอมพิวเตอร์

  1. ระบบ Windows: กดปุ่ม Windows + R บนแป้นพิมพ์ พิมพ์ cmd แล้วกด Enter
  2. พิมพ์คำสั่ง ipconfig แล้วกด Enter
  3. เลื่อนหาหัวข้อ Wireless LAN adapter Wi-Fi (หากใช้ Wi-Fi) หรือ Ethernet adapter (หากใช้สายแลน)
  4. จดหมายเลขในบรรทัด IPv4 Address เช่น 192.168.1.50 (นี่คือ IP คอมพิวเตอร์ของคุณ)
  5. ระบบ macOS: ไปที่ System Settings > Network แล้วเลือกเครือข่ายที่ใช้งานเพื่อดู IP

ฝั่งโดรน (DJI App) 3. การใส่ IP ในแอปโดรน DJI

  1. เปิดแอปควบคุมโดรน (เช่น DJI Fly หรือ DJI Pilot) ➡️ เข้าสู่โหมดสตรีมมิ่งสด Live Stream / Live Broadcast
  2. เลือกสตรีมมิ่งผ่านโปรโตคอล RTMP
  3. ใส่ช่องกรอก URL สตรีมด้วย IP ของคอมพิวเตอร์ที่จดไว้จากข้อ 2 ตามโครงสร้างดังนี้:
    rtmp://[IP คอมพิวเตอร์]/live/drone
    (เช่น: rtmp://192.168.1.50/live/drone)
  4. กดเริ่มสตรีมในแอปโดรน

การควบคุม 4. เชื่อมโยงสตรีมเข้า OBS

  1. บนมือถือ/รีโมทที่เปิดหน้าบราวเซอร์ควบคุมนี้ ให้กรอกข้อมูลการตั้งค่าเชื่อมต่อ OBS (ใช้เลข IP คอมพิวเตอร์ข้อ 2 และรหัสผ่าน)
  2. นำลิงก์ RTMP เดียวกับที่ใส่ในโดรน DJI (ข้อ 3) มาใส่ในช่อง **ลิงก์สตรีมวิดีโอ** (หรือสแกน QR Code จากโดรน)
  3. กดปุ่ม **"ส่งภาพฉายใน OBS"** เพื่อส่งสตรีมเข้าไปเล่นใน OBS ทันที!

💡 เคล็ดลับความสะดวก: คุณสามารถกดปุ่ม **"สร้างลิงก์ด่วน"** หลังจากตั้งค่าบนคอมพิวเตอร์ครั้งแรกเสร็จ แล้วส่งลิงก์นั้นไปเปิดในมือถือ หน้าจอมือถือจะจำการตั้งค่าเชื่อมต่อทั้งหมดอัตโนมัติ ไม่ต้องกรอกซ้ำครับ
