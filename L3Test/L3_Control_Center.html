📖 วิธีใช้งาน

1. Import Excel
กดแถบ "📂 Import Excel Data" ด้านล่าง แล้วเลือกหรือลากไฟล์
Columns: ID, name, LabelName1, LabelName2, LabelName3, image

2. ตั้งค่า Cards
กดปุ่ม "⚙️ ตั้งค่า Cards" เพื่อเพิ่ม/ลบ/แก้ไข Cards
- Single: แสดงข้อมูล 1 คน
- Double: แสดงข้อมูล 2 คน (Home/Away)

3. สร้าง OBS Sources
กดปุ่ม ⚙️ ที่มุมขวาบนของแต่ละ Card
กดที่ชื่อ Group (มี ➕) เพื่อสร้าง Sources อัตโนมัติ
ระบบจะสร้าง Text และ Image Sources ใน Scene ของ OBS

4. ชื่อ Source ใน OBS
Single Card:
T_Name_1, T_Label1_1, I_Avatar_1
Double Card:
T_Name_Home_2, T_Name_Away_2

5. Group Sources ใน OBS
หลังจากสร้าง Sources แล้ว:
1. เลือก Sources ทั้งหมด (กด Ctrl ค้าง)
2. คลิกขวา → Group Selected Items
3. ตั้งชื่อ Group ตามที่แสดง เช่น G_LowerThird_1

6. วิธีใช้งาน Browser Source (🎨 ทางเลือกสำหรับเล่นแอนิเมชัน)
1. เพิ่ม Browser Source ใน OBS (ตั้งชื่อตามต้องการ)
2. ใส่ URL ของหน้าจอแสดงผลกราฟิก (ไม่ต้องติ๊กเลือก Local file):

3. ตั้งค่าขนาดกว้าง 1920 สูง 1080
4. เปิดใช้งานกราฟิกที่เมนูตั้งค่าของแต่ละ Card (⚙️ → ติ๊ก "เปิดใช้งานกราฟิกสำหรับ Card นี้")
