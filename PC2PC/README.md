# NDI Audio Bridge

Stream audio (Mic + Desktop) over NDI network.

## Requirements

1. **Python 3.8+** - [Download](https://www.python.org/downloads/)
2. **NDI Runtime** - [Download](https://ndi.video/tools/)
3. **VB-Cable** (optional) - [Download](https://vb-audio.com/Cable/)

## Quick Start

1. Double-click `Run.bat`
2. Choose **Sender** or **Receiver** mode
3. Select devices and click Start
